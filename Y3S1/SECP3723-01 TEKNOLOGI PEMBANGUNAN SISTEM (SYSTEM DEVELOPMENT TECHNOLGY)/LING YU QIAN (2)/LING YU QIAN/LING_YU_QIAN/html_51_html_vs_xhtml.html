HTML vs. XHTML

HTML5 is forgiving: closing tags can sometimes be omitted and attribute values may be unquoted in some cases. XHTML required stricter XML rules.

HTML5 example (flexible): <br>

XHTML-style self-closing (XML): <img src="img.png" alt="x" />

Recommendation: Use HTML5 with a proper <meta charset="utf-8"> and valid markup.
