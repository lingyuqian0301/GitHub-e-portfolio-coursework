Table Headers

Use the TH element to define table headers.

Firstname Lastname Age
Yu Qian Ling 20
Jolyn Tee 21
Kuganes Shavindran 25

Vertical Table Headers

The first column becomes table headers if you set the first table cell in each table row to a TH element:

Firstname Yu Qian Jolyn
Lastname Ling Tee
Age 20 21

Text-align Headers

To left-align/center-align/right-align the table headers, use the CSS text-align property.

Firstname Lastname Age
Yu Qian Ling 20
Jolyn Tee 21

A header that spans two columns

Use the colspan attribute to have a header span over multiple columns.

Name Age
Yu Qian Ling 20
Jolyn Tee 21

Table Caption

To add a caption to a table, use the caption tag.

Month Savings
